# Employment Types

Define and manage employment type classifications within the organization.

## Overview

Employment types define the various contract arrangements under which employees are engaged:

| Type       | Description                   |
| ---------- | ----------------------------- |
| Full-Time  | Standard full-time employment |
| Part-Time  | Reduced hours employment      |
| Contract   | Fixed-term contract           |
| Freelance  | Independent contractor        |
| Internship | Intern position               |

## Configuration

1. Navigate to **Settings** → **Employment Types**
2. Add, edit, or remove employment type definitions
3. Assign employment types to employee records
