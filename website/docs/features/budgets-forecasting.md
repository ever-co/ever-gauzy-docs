# Budgets & Forecasting

Track project budgets and forecast financial performance.

## Project Budgets

### Setting a Budget

1. Open a project → **Settings** → **Budget**
2. Configure:
   - Budget type: Hours or Cost
   - Budget amount
   - Alert thresholds (50%, 75%, 90%, 100%)

### Budget Tracking

| Metric          | Description                     |
| --------------- | ------------------------------- |
| Budget          | Total allocated amount          |
| Spent           | Used so far (hours or cost)     |
| Remaining       | Budget - Spent                  |
| Burn rate       | Average spending per day        |
| Projected total | Estimated total at current rate |

### Budget Alerts

Notifications at configurable thresholds:

- 🟢 < 50% — On track
- 🟡 50-75% — Moderate usage
- 🟠 75-90% — High usage
- 🔴 > 90% — Near limit

## Revenue Forecasting

| Report                | Insight                        |
| --------------------- | ------------------------------ |
| Monthly Revenue       | Trend over months              |
| Project Profitability | Revenue vs cost per project    |
| Client Revenue        | Revenue breakdown by client    |
| Employee Utilization  | Billable vs non-billable hours |
| Pipeline Forecast     | Expected revenue from deals    |

## Budget Types

| Type        | Unit | Calculation           |
| ----------- | ---- | --------------------- |
| Hours-based | hrs  | Sum of time logged    |
| Cost-based  | $    | Hours × employee rate |
| Fixed price | $    | Against project value |
