# Screening Tasks

Assign and track evaluation tasks during the candidate hiring process.

## Overview

Screening tasks are used by recruiters and hiring managers to evaluate candidates through structured assessments — coding challenges, personality tests, or review tasks.

## Creating a Screening Task

| Field       | Description                              |
| ----------- | ---------------------------------------- |
| Title       | Test name (e.g., "Technical Assessment") |
| Description | Instructions for the candidate           |
| Due Date    | Completion deadline                      |
| Candidate   | Associated candidate                     |
| Status      | `TODO`, `IN_PROGRESS`, `COMPLETED`       |

## Tracking Results

- Track completion status per candidate
- Record pass/fail results
- Link screening results to candidate feedback
