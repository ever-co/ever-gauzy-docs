# CRM Overview

The Customer Relationship Management (CRM) module helps manage client relationships, sales pipelines, and business contacts.

## CRM Modules

```
CRM Platform
├── Contacts Management   ← Clients, leads, customers
├── Sales Pipelines       ← Deal stages and tracking
├── Proposals             ← Client proposals
├── Email Integration     ← Communication tracking
└── Reports               ← Sales analytics
```

## Contact Types

| Type         | Description        | Use Case          |
| ------------ | ------------------ | ----------------- |
| **Lead**     | Potential customer | Initial contact   |
| **Client**   | Active customer    | Ongoing business  |
| **Customer** | Paying customer    | Transaction-based |

## Key Features

- **Contact profiles** with company details, emails, phones
- **Pipeline management** with customizable stages
- **Deal tracking** with value and probability
- **Communication history** linked to contacts
- **Revenue forecasting** from pipeline data
- **Dashboard** with conversion metrics
