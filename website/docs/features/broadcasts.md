# Broadcasts

Send organization-wide announcements and broadcast messages to all employees.

## Overview

Broadcasts enable administrators to send important messages to all members of an organization — used for announcements, policy updates, and company-wide communication.

## Creating a Broadcast

1. Navigate to **Organization** → **Broadcasts**
2. Click **New Broadcast**
3. Enter the title and message
4. Select the broadcast type
5. Submit to send to all organization members
