# Recurring Expenses

Manage automated recurring expense entries for both organizations and employees.

## Overview

Recurring expenses automate the tracking of regular costs like rent, subscriptions, salaries, and utilities.

## Organization Recurring Expenses

| Field         | Description                |
| ------------- | -------------------------- |
| Category Name | Expense category           |
| Value         | Recurring amount           |
| Currency      | Currency code              |
| Start Date    | First occurrence           |
| End Date      | Last occurrence (optional) |
| Split Expense | Split across members       |

## Employee Recurring Expenses

Similar to organization expenses but scoped to individual employees:

- Monthly allowances
- Insurance premiums
- Commuting expenses
