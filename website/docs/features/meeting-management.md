# Meeting Management

Schedule and track team meetings.

## Creating Meetings

1. Go to **Calendar** → **Schedule Meeting**
2. Configure:
   - Title and agenda
   - Date, start time, end time
   - Recurrence (daily, weekly, monthly)
   - Attendees (employees)
   - Location or video link
   - Reminder (5min, 15min, 30min, 1hr)

## Meeting Types

| Type            | Use Case                  |
| --------------- | ------------------------- |
| Standup         | Daily team sync           |
| Sprint Planning | Plan sprint work          |
| Retrospective   | Sprint review             |
| 1-on-1          | Manager/employee check-in |
| All Hands       | Company-wide update       |
| Client Call     | External client meeting   |

## Calendar Integration

Meetings sync with:

- Google Calendar
- Microsoft Outlook
- iCal/CalDAV
- Desktop calendar

## Meeting Time Tracking

Link meetings to time tracking:

1. Start timer during meeting
2. Associate with project/task
3. Time automatically logged
4. Shows in timesheet

## Recurring Meetings

Set up recurring patterns:

```
Every Monday at 9:00 AM — Team Standup
Every other Friday at 2:00 PM — Sprint Review
First Monday monthly — All Hands
```
