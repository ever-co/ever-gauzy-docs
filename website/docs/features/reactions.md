# Reactions

Emoji reactions on comments, tasks, and other entities.

## Overview

Users can add emoji reactions to comments and entities to express feedback without writing a full comment — similar to reactions in Slack or GitHub.

## Adding Reactions

1. Hover over a comment or entity
2. Click the reaction button (😀)
3. Select an emoji
4. Your reaction is added and visible to all team members

## Removing Reactions

Click your existing reaction again to remove it.

## Supported Entities

| Entity  | Description               |
| ------- | ------------------------- |
| Comment | React to discussion items |
| Task    | React to tasks            |
