# Organization Teams

Create and manage cross-functional teams.

## Overview

Teams group employees for project collaboration, goals tracking, and task assignment.

## Creating Teams

1. Go to **Organization** → **Teams**
2. Click **Add Team**
3. Fill in:
   - Team name
   - Description
   - Members (select employees)
   - Manager(s)
   - Tags
4. Save

## Team Types

| Type             | Description               |
| ---------------- | ------------------------- |
| Department       | Organizational unit teams |
| Project          | Project-specific teams    |
| Cross-Functional | Multi-department teams    |
| Temporary        | Time-limited task forces  |

## Team Features

- **Shared tasks** — assign tasks to team
- **Team goals** — set and track team OKRs
- **Time reports** — aggregate time by team
- **Activity feed** — team activity stream
- **Team dashboard** — team-specific metrics

## Team Management

| Action         | Access Required |
| -------------- | --------------- |
| Create team    | Admin, Manager  |
| Add members    | Team manager    |
| Remove members | Team manager    |
| Delete team    | Admin           |
| View team data | Team members    |
