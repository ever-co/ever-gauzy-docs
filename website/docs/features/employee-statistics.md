# Employee Statistics & Analytics

View and analyze employee productivity metrics.

## Overview

Employee statistics provide insights into:

- Total work hours per period
- Income and expense tracking per employee
- Bonus and salary summaries
- Activity levels and productivity metrics

## Available Statistics

| Metric           | Description                    |
| ---------------- | ------------------------------ |
| Total Work Hours | Hours tracked in time period   |
| Monthly Count    | Active employees per month     |
| Total Income     | Revenue attributed to employee |
| Total Expenses   | Expenses incurred by employee  |
| Total Bonuses    | Bonus payments received        |

## Viewing Statistics

### Organization-Level

1. Navigate to **Dashboard** → **Employee Statistics**
2. Select date range
3. View aggregate statistics

### Per-Employee

1. Navigate to **Employees** → select employee
2. Click **Statistics** tab
3. View individual metrics

## API

See [Employee Sub-Resource Endpoints](../api/employee-sub-resource-endpoints) for the statistics API.

## Charts and Visualizations

- **Bar Chart** — Hours by month
- **Pie Chart** — Income distribution by project
- **Line Chart** — Productivity trends over time
