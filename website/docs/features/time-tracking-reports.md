# Time Tracking Reports

Detailed time tracking reports and analytics.

## Report Types

| Report              | Description                   |
| ------------------- | ----------------------------- |
| Daily Time Report   | Hours per day per employee    |
| Weekly Summary      | Weekly aggregate by employee  |
| Project Time Report | Hours by project              |
| Client Time Report  | Billable hours per client     |
| Activity Summary    | Activity levels and app usage |
| Timesheet Status    | Approval status across org    |

## Generating Reports

1. Go to **Time Tracking** → **Reports**
2. Select report type
3. Configure filters:
   - Date range
   - Employees
   - Projects/clients
   - Teams/departments
4. Click **Generate**
5. Export as PDF, CSV, or Excel

## Key Metrics

### Employee Productivity

| Metric           | Calculation                   |
| ---------------- | ----------------------------- |
| Total Hours      | Sum of all time logs          |
| Billable Hours   | Hours marked as billable      |
| Non-Billable     | Total - Billable              |
| Utilization Rate | Billable / Expected × 100     |
| Average Activity | Avg activity level percentage |

### Project Metrics

```
Project Hours = Sum of time logs for project
Budget Consumed = Project Hours / Budget Hours × 100
Employee Cost = Hours × Employee Rate
Revenue = Hours × Client Billing Rate
Profit = Revenue - Employee Cost
```

## Scheduled Reports

Set up automatic report delivery:

1. **Settings** → **Scheduled Reports**
2. Select report type
3. Set schedule (daily, weekly, monthly)
4. Add recipients
5. Save
