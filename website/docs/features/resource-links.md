# Resource Links

Attach external URLs and resources to any entity (tasks, projects, etc.).

## Overview

Resource links allow users to attach relevant external resources (documentation, design files, wikis, external tools) to any platform entity for easy reference.

## Adding a Resource Link

1. Open an entity detail view (task, project, etc.)
2. Navigate to the **Resources** section
3. Click **Add Resource Link**
4. Enter the title and URL

## Use Cases

- Link design mockups (Figma, Sketch) to tasks
- Attach specification documents
- Reference external issue trackers
- Link to deployment dashboards
