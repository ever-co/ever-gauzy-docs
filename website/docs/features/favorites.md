# Favorites

Bookmark frequently accessed entities for quick navigation.

## Overview

The Favorites feature lets users bookmark tasks, projects, contacts, teams, and employees for one-click access from the sidebar or dashboard.

## Supported Entities

| Entity              | Description               |
| ------------------- | ------------------------- |
| Task                | Quick access to key tasks |
| OrganizationProject | Pin important projects    |
| OrganizationContact | Pin CRM contacts          |
| OrganizationTeam    | Pin teams                 |
| Employee            | Pin team members          |

## Managing Favorites

### Adding a Favorite

Click the ⭐ (star) icon on any supported entity to add it to your favorites.

### Removing a Favorite

Click the filled star icon again to remove from favorites.

### Viewing Favorites

Access your favorites from the sidebar or dashboard widget for quick navigation.
