# Organization Project Settings

Configure project management settings at the organization level.

## Project Settings

1. Go to **Settings** → **Projects**
2. Configure:

### Task Statuses

Define custom task status workflows:

| Status      | Default | Customizable |
| ----------- | ------- | ------------ |
| OPEN        | ✅      | ✅           |
| IN_PROGRESS | ✅      | ✅           |
| IN_REVIEW   | ❌      | ✅           |
| DONE        | ✅      | ✅           |
| BLOCKED     | ❌      | ✅           |
| RELEASED    | ❌      | ✅           |

### Issue Types

| Type    | Default | Icon   |
| ------- | ------- | ------ |
| TASK    | ✅      | ✅     |
| BUG     | ✅      | 🐛     |
| STORY   | ✅      | 📖     |
| EPIC    | ✅      | ⭐     |
| FEATURE | ❌      | Custom |

### Priority Levels

See [Task Priorities & Sizes](./task-priorities-sizes).

### Project Modules

Enable/disable project features:

| Module       | Description             |
| ------------ | ----------------------- |
| Sprints      | Agile sprint management |
| Kanban Board | Visual task board       |
| Gantt Chart  | Timeline view           |
| Time Logging | Project time tracking   |
| Budget       | Project budget tracking |
| Wiki         | Project documentation   |

## Project Templates

Create reusable project templates:

1. Go to **Settings** → **Project Templates**
2. Click **Create Template**
3. Define default tasks, statuses, and modules
4. Use when creating new projects
