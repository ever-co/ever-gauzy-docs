# Knowledge Base

Build an internal knowledge base and help center.

## Overview

The Knowledge Base plugin provides:

- Categorized articles
- Rich text editor
- Search functionality
- Public and private articles

## Setting Up

### Enable Plugin

The Knowledge Base is a built-in plugin. Enable it in **Settings** → **Features**.

### Create Categories

1. Go to **Knowledge Base** → **Categories**
2. Add categories:
   - Getting Started
   - How-To Guides
   - Policies
   - FAQ

### Write Articles

1. Go to **Knowledge Base** → **Articles**
2. Click **New Article**
3. Select category
4. Write content using the rich text editor
5. Set visibility (Public/Private)
6. Publish

## Article Structure

| Field      | Description              |
| ---------- | ------------------------ |
| Title      | Article headline         |
| Category   | Organizing category      |
| Content    | Rich text content        |
| Status     | Draft or Published       |
| Visibility | Public or Private        |
| Author     | Auto-set to current user |

## Search

Users can search across all published articles using the search bar.

## Use Cases

- Employee onboarding guides
- Company policies and procedures
- Technical documentation
- FAQ for common questions
