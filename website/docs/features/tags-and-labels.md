# Tags & Labels

Flexible tagging system for categorizing and filtering entities across the platform.

## Overview

Tags provide a cross-cutting categorization mechanism. Any entity (tasks, employees, projects, contacts, products, etc.) can be tagged for easy filtering and grouping.

## Tag Properties

| Property    | Description                    |
| ----------- | ------------------------------ |
| Name        | Tag display name               |
| Color       | Visual color code              |
| Description | Optional description           |
| Icon        | Optional icon                  |
| Is System   | System-defined tag (immutable) |
| Tag Type    | Category/type of tag           |

## Tag Types

Tag Types provide a hierarchical categorization:

| Example Type | Used For               |
| ------------ | ---------------------- |
| Skills       | Employee skills        |
| Technology   | Tech stack tags        |
| Department   | Department labels      |
| Priority     | Custom priority labels |

## Using Tags

### Adding Tags

Tags can be added from any entity's edit view using a searchable tag picker.

### Filtering by Tags

Most list views support filtering by tags — select one or more tags to narrow results.

### Tag Management

Manage organizational tags from **Settings** → **Tags**.
