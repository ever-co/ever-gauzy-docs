# ERP Overview

The Enterprise Resource Planning (ERP) module handles financial operations including invoicing, expense tracking, payments, and income management.

## ERP Modules

```
ERP Platform
├── Invoicing           ← Create and send invoices
├── Estimates           ← Client proposals with pricing
├── Expenses            ← Track business expenses
├── Payments            ← Record payment receipts
├── Income              ← Track revenue streams
├── Recurring Expenses  ← Auto-generated periodic expenses
└── Financial Reports   ← Revenue, profit, and cash flow
```

## Financial Dashboard

The ERP dashboard provides at-a-glance metrics:

| Metric          | Description             |
| --------------- | ----------------------- |
| **Revenue**     | Total income received   |
| **Expenses**    | Total expenses recorded |
| **Profit**      | Revenue minus expenses  |
| **Outstanding** | Unpaid invoice total    |
| **Overdue**     | Overdue invoice total   |

## Currency Support

Ever Gauzy supports multiple currencies with:

- Organization-level default currency
- Per-invoice currency override
- Per-expense currency tracking
- Exchange rate awareness
