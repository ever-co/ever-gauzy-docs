# Team Collaboration Features

In-app collaboration tools for teams.

## Comments & Mentions

### Commenting on Tasks

1. Open any task
2. Scroll to **Comments** section
3. Type your comment
4. Mention team members with `@`
5. Submit — mentioned users are notified

### @Mentions

- `@user` — notify a specific person
- `@team` — notify a whole team
- Mentions trigger in-app and email notifications

## Reactions

React to comments with emoji:

- 👍 Thumbs up
- ❤️ Heart
- 🎉 Celebration
- 👀 Eyes (acknowledging)

## Entity Subscriptions

Subscribe to entities (tasks, projects) for updates:

1. Open any entity
2. Click **Subscribe** (bell icon)
3. Receive notifications for changes:
   - Status updates
   - New comments
   - Assignee changes
   - Due date changes

## Team Activity Feed

Dashboard shows real-time team activity:

- Who started/stopped timers
- Task status changes
- New comments
- Timesheet submissions
- Employee joins/leaves

## Resource Links

Attach links to any entity:

- Documentation URLs
- Design files (Figma, Sketch)
- Related tickets
- External resources
