# Custom Views

Save and reuse custom filter configurations for task lists, employee lists, and other data views.

## Overview

Custom views allow users to save frequently used filter, sort, and group configurations for quick access. Each view stores the exact state of filters and column visibility.

## Available View Types

| View Type | Applies To         |
| --------- | ------------------ |
| Kanban    | Tasks by status    |
| Sprint    | Sprint task boards |
| List      | Table views        |
| Grid      | Card-based layouts |

## Creating a View

1. Configure filters, sorting, and grouping as desired
2. Click **Save View**
3. Name the view
4. The view appears in the sidebar for quick access

## View Configuration

Each view stores:

| Setting       | Description            |
| ------------- | ---------------------- |
| Filters       | Active filter criteria |
| Sort Order    | Column sort direction  |
| Column Layout | Visible columns        |
| Group By      | Grouping criteria      |
