# Interview Scheduling

Schedule and manage candidate interviews.

## Creating Interviews

1. Open a candidate profile
2. Go to **Interviews** tab
3. Click **Schedule Interview**
4. Fill in:
   - Interview title
   - Date and time
   - Duration
   - Interviewers (select employees)
   - Interview type (video, phone, in-person)
   - Location or video link
5. Save — invitations sent automatically

## Interview Types

| Type         | Description                 |
| ------------ | --------------------------- |
| Phone Screen | Initial 15-30 min call      |
| Technical    | Coding/technical assessment |
| Behavioral   | Situational questions       |
| Panel        | Multiple interviewers       |
| Final        | Final round with leadership |

## Interview Panel

Assign multiple interviewers:

| Role             | Responsibility             |
| ---------------- | -------------------------- |
| Lead Interviewer | Moderates the session      |
| Technical        | Evaluates technical skills |
| HR               | Evaluates culture fit      |
| Hiring Manager   | Final decision maker       |

## Feedback Collection

After the interview:

1. Each interviewer submits feedback
2. Rate candidate on criteria
3. Add notes and observations
4. Submit hire/no-hire recommendation
5. All feedback aggregated on candidate profile

## Integration

Interviews sync with:

- Google Calendar
- Email notifications
- Desktop calendar apps (iCal)
