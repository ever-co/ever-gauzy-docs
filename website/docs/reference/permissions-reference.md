# Permissions Reference

Complete list of all permissions in Ever Gauzy.

## User Management

| Permission                 | Description                  |
| -------------------------- | ---------------------------- |
| `ORG_USERS_VIEW`           | View users                   |
| `ORG_USERS_EDIT`           | Edit users                   |
| `CHANGE_ROLES_PERMISSIONS` | Manage roles and permissions |
| `ADMIN_EDIT_DELETE`        | Admin-level edit and delete  |
