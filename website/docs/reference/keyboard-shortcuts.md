# Keyboard Shortcuts

Quick keyboard shortcuts for navigating and using Ever Gauzy.

## Global Shortcuts

| Shortcut               | Action                     |
| ---------------------- | -------------------------- |
| `Ctrl/Cmd + K`         | Quick search / command bar |
| `Ctrl/Cmd + /`         | Toggle sidebar             |
| `Ctrl/Cmd + Shift + N` | New task                   |

## Time Tracking

| Shortcut               | Action           |
| ---------------------- | ---------------- |
| `Ctrl/Cmd + Shift + T` | Start/stop timer |
| `Ctrl/Cmd + Shift + S` | Take screenshot  |

## Navigation

| Shortcut       | Action              |
| -------------- | ------------------- |
| `Ctrl/Cmd + 1` | Go to Dashboard     |
| `Ctrl/Cmd + 2` | Go to Tasks         |
| `Ctrl/Cmd + 3` | Go to Projects      |
| `Ctrl/Cmd + 4` | Go to Time Tracking |
| `Ctrl/Cmd + 5` | Go to Employees     |

## Task Management

| Shortcut      | Action             |
| ------------- | ------------------ |
| `Enter`       | Open selected task |
| `Escape`      | Close dialog/modal |
| `Tab`         | Next field         |
| `Shift + Tab` | Previous field     |

## Desktop App

| Shortcut               | Action           |
| ---------------------- | ---------------- |
| `Ctrl/Cmd + Shift + G` | Show/hide app    |
| `Ctrl/Cmd + Q`         | Quit application |
