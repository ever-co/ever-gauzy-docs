# Admin Dashboard Guide

Navigate and use the Ever Gauzy admin dashboard for system management.

## Overview

The admin dashboard provides centralized management for:

- **Organizations** — create and manage business units
- **Employees** — manage team members and their settings
- **Roles & Permissions** — configure access control
- **Feature Flags** — enable/disable platform features
- **System Settings** — configure global settings
- **Reports** — access analytics and reports

## Dashboard Navigation

### Sidebar Menu

| Section       | Description                       |
| ------------- | --------------------------------- |
| Dashboard     | Overview metrics and widgets      |
| Time Tracking | Timesheets, activity, screenshots |
| Accounting    | Invoices, expenses, payments      |
| Projects      | Projects, tasks, sprints          |
| Contacts      | CRM contacts and pipelines        |
| Employees     | Employee management               |
| Organization  | Org settings and departments      |
| Goals         | Goals, KPIs, and OKRs             |
| Approvals     | Time off and expense approvals    |
| Reports       | Built-in reports                  |
| Settings      | System configuration              |

### Dashboard Widgets

The main dashboard displays configurable widgets:

| Widget           | Description              |
| ---------------- | ------------------------ |
| Today's Log      | Current day time log     |
| Active Tasks     | Tasks in progress        |
| Team Activity    | Team members' status     |
| Income/Expense   | Financial summary        |
| Project Progress | Project completion stats |

## Quick Actions

| Action            | Path                   |
| ----------------- | ---------------------- |
| Add Employee      | Employees → Add        |
| Create Project    | Projects → Add         |
| Create Invoice    | Accounting → Invoices  |
| Start Timer       | Time Tracking → Timer  |
| Approve Timesheet | Approvals → Timesheets |
