# Organization Setup Guide

Step-by-step guide to setting up a new organization in Ever Gauzy.

## Prerequisites

- Active tenant with admin access
- Organization details ready

## Step 1: Create the Organization

1. Navigate to **Organization** → **Add Organization**
2. Fill in the details:

| Field             | Description             |
| ----------------- | ----------------------- |
| Organization Name | Your company/team name  |
| Currency          | Default currency        |
| Date Format       | Preferred date format   |
| Time Zone         | Organization time zone  |
| Start Week On     | First day of week       |
| Region Code       | Region for localization |

## Step 2: Configure Departments

1. Go to **Organization** → **Departments**
2. Create departments (e.g., Engineering, Marketing, Sales)
3. Add department tags

## Step 3: Add Positions

1. Go to **Organization** → **Positions**
2. Create positions (e.g., Developer, Designer, Manager)

## Step 4: Configure Projects

1. Navigate to **Projects** → **Add Project**
2. Set billing type, budget, and team members
3. Define task statuses, priorities, and sizes

## Step 5: Invite Team Members

1. Navigate to **Employees** → **Invite**
2. Enter email addresses
3. Assign roles and departments
4. Send invitations

## Step 6: Configure Integrations

1. Go to **Settings** → **Integrations**
2. Connect with GitHub, GitLab, Jira, etc.
3. Configure sync settings

## Step 7: Set Feature Flags

1. Go to **Settings** → **Feature Toggles**
2. Enable/disable features based on your needs

## Post-Setup Checklist

- [ ] Organization details configured
- [ ] Departments and positions created
- [ ] At least one project created
- [ ] Team members invited
- [ ] Time tracking settings configured
- [ ] Email templates customized
- [ ] Feature flags reviewed
