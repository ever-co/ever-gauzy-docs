# Información de Soporte

## Aplicación de Gestión Empresarial

## Introducción
La Aplicación de Gestión Empresarial es una herramienta poderosa diseñada para ayudar a las empresas a gestionar de manera efectiva a sus empleados y horarios de trabajo. Proporciona una plataforma centralizada para gestionar la información de los empleados, realizar un seguimiento de sus actividades, así como planificar y supervisar las horas de trabajo.

## Funcionalidades
La Aplicación de Gestión Empresarial ofrece una amplia gama de funcionalidades para facilitar la gestión de empleados y horarios de trabajo. Algunas de las características clave incluyen:

- Gestión de Empleados: Permite almacenar y gestionar información detallada de cada empleado, incluyendo sus datos de contacto, rol, departamento, estado laboral, etc.
- Gestión de Horarios de Trabajo: Facilita la planificación de horarios de trabajo de los empleados, permitiendo definir franjas horarias, días libres, días festivos, etc.
- Seguimiento de Tiempo: Permite a los empleados ingresar sus horas de trabajo diarias, solicitar permisos o días libres, y permite a los gerentes aprobar estas solicitudes.
- Informes y Estadísticas: Genera informes detallados sobre las horas de trabajo de los empleados, ausencias, permisos, horas extras, etc., para facilitar el análisis y la toma de decisiones.

## Uso
La Aplicación de Gestión Empresarial ofrece una interfaz fácil de usar e intuitiva para facilitar su uso. Aquí hay algunos pasos básicos para comenzar:

1. Inicia sesión en la aplicación utilizando tus credenciales de administrador.

2. Ve a la sección "Gestión de Empleados" para agregar, modificar o eliminar empleados de la base de datos.

3. Utiliza la sección "Gestión de Horarios de Trabajo" para crear y gestionar los horarios de trabajo de los empleados, teniendo en cuenta los días festivos y las ausencias.

4. Los empleados pueden iniciar sesión en la aplicación y utilizar la funcionalidad "Seguimiento de Tiempo" para registrar sus horas de trabajo diarias, solicitar permisos o informar problemas.

5. Los gerentes pueden aprobar o rechazar las solicitudes de permisos y descanso de los empleados, y generar informes sobre las horas de trabajo y las ausencias.
