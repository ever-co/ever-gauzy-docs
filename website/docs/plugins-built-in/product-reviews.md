# Product Reviews Plugin

Enable product reviews and ratings for the inventory module.

## Overview

The Product Reviews plugin adds customer/user review capabilities to products.

## Features

| Feature           | Description              |
| ----------------- | ------------------------ |
| Star ratings      | 1-5 star rating system   |
| Text reviews      | Written product feedback |
| Review moderation | Approve/reject reviews   |
| Average rating    | Auto-calculated averages |

## API

- `GET /api/product-reviews` — list reviews
- `POST /api/product-reviews` — create review
- `PUT /api/product-reviews/:id` — update review
- `DELETE /api/product-reviews/:id` — delete review
