# Job Board Plugins

Plugins for job posting, search, proposals, employee matching, and external board integration.

## Job Search Plugin

Aggregates job postings from multiple external platforms.

### Features

- Search across multiple job boards
- Filter by technology, location, and type
- Save searches and get notifications
- Apply directly from within Gauzy

### Configuration

| Variable                    | Description        |
| --------------------------- | ------------------ |
| `GAUZY_AI_GRAPHQL_ENDPOINT` | AI service URL     |
| `GAUZY_AI_API_KEY`          | AI service API key |

## Job Proposal Plugin

Manage proposals sent to clients for freelance/contract work.

### Features

- Create and track proposals
- Link proposals to jobs and contacts
- Track acceptance/rejection rates
- Generate proposal documents from templates

## Job Employee Plugin

Match employees to job openings based on skills and availability.

### Features

- Employee skill matching
- Availability checking
- Automatic recommendations
- Apply/reject workflow

## Job Matching Plugin

AI-powered job matching using the Gauzy AI service.

### Features

- Skill-based matching algorithm
- Experience weighting
- Location preferences
- Salary range matching
