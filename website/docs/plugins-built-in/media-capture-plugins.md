# Media Capture Plugins

Desktop application plugins for capturing screenshots, camera shots, and audio.

## Camshot Plugin

Captures webcam photos at configurable intervals during time tracking.

### Features

- Periodic webcam capture
- Privacy controls (employee can disable)
- Stored with time slots for activity verification

### Configuration

| Variable              | Description               | Default |
| --------------------- | ------------------------- | ------- |
| `SCREENSHOTS_ENABLED` | Enable screenshot capture | `true`  |

## Soundshot Plugin

Records audio snippets during time tracking sessions.

### Features

- Short audio recordings at intervals
- Audio quality configuration
- Privacy controls

## Videos Plugin

Records screen activity video during time tracking.

### Features

- Screen recording during time slots
- Configurable duration and quality
- Stored alongside screenshots and time data
