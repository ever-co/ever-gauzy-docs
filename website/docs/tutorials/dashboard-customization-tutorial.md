# Tutorial: Dashboard Customization

Personalize your dashboard with widgets and layouts.

## Step 1: Access Dashboard Settings

1. Navigate to the **Dashboard**
2. Click the **Gear (⚙)** icon in the top right

## Step 2: Add Widgets

Available widgets:

| Widget               | Shows                     |
| -------------------- | ------------------------- |
| Today's Tracked Time | Hours tracked today       |
| Team Activity        | Who's online and tracking |
| Income/Expenses      | Financial summary         |
| Outstanding Invoices | Unpaid invoices           |
| Active Tasks         | In-progress tasks         |
| Project Progress     | Completion percentages    |
| Employee Count       | Total active employees    |

To add:

1. Click **Add Widget**
2. Select widget type
3. Choose position and size

## Step 3: Customize Layout

- **Drag** widgets to reorder
- **Resize** by dragging edges or corners
- **Remove** by clicking the × button

## Step 4: Set Date Range

The dashboard respects the global date range picker:

- Today
- This Week
- This Month
- Custom Range

## Step 5: Save Layout

Your dashboard layout is saved automatically per user.
