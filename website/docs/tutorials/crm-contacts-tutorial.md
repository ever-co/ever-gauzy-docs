# Tutorial: CRM & Contacts

Manage clients, leads, and customer relationships.

## Step 1: Add a Contact

1. Navigate to **CRM** → **Contacts**
2. Click **Add Contact**
3. Choose type:
   - **Client** — paying customers
   - **Lead** — potential customers
   - **Customer** — general contacts
4. Fill in:
   - Name, email, phone
   - Company/organization
   - Address
   - Tags
5. Click **Save**

## Step 2: Create a Pipeline

Pipelines track deals through stages.

1. Go to **CRM** → **Pipelines**
2. Click **Add Pipeline**
3. Name it (e.g., "Sales Pipeline")
4. Define stages:

| Stage         | Probability |
| ------------- | ----------- |
| Prospecting   | 10%         |
| Qualification | 25%         |
| Proposal      | 50%         |
| Negotiation   | 75%         |
| Closed Won    | 100%        |
| Closed Lost   | 0%          |

## Step 3: Add Deals

1. Go to **CRM** → **Deals**
2. Click **Add Deal**
3. Fill in:
   - Deal title and value
   - Associated contact
   - Pipeline and stage
   - Expected close date
4. Click **Save**

## Step 4: Track Progress

- Drag deals between pipeline stages
- Add notes and activities
- Set follow-up reminders
- View revenue forecasts

## Step 5: Generate Reports

1. Go to **Reports** → **CRM Reports**
2. View:
   - Pipeline value by stage
   - Win/loss rates
   - Revenue forecasts
   - Activity history
