# Tutorial: Setting Up Your First Organization

Create and configure an organization in Ever Gauzy.

## What You'll Learn

- Create a new organization
- Configure basic settings (currency, timezone, date format)
- Set up departments and positions
- Invite team members

## Step 1: Create Organization

1. After login, go to **Settings** → **Organizations**
2. Click **Add Organization**
3. Fill in:
   - **Organization Name** — your company name
   - **Currency** — default currency (e.g., USD)
   - **Date Type** — preferred date format
   - **Time Zone** — your timezone
4. Click **Save**

## Step 2: Upload Logo

1. In the organization settings, click the avatar area
2. Upload your company logo (recommended: 256×256px PNG)
3. Save changes

## Step 3: Configure Date & Time

| Setting           | Recommended    | Options            |
| ----------------- | -------------- | ------------------ |
| Date Format       | `MM/DD/YYYY`   | Various formats    |
| Time Format       | `12h` or `24h` | 12-hour or 24-hour |
| Start of Week     | Monday         | Monday or Sunday   |
| Fiscal Year Start | January        | Any month          |

## Step 4: Set Up Departments

1. Go to **Organization** → **Departments**
2. Click **Add Department**
3. Create departments:
   - Engineering
   - Marketing
   - Sales
   - Operations
4. Add managers to each department

## Step 5: Create Positions

1. Go to **Organization** → **Positions**
2. Add common positions:
   - Software Engineer
   - Project Manager
   - Designer
   - Marketing Specialist

## Step 6: Invite Team Members

1. Go to **Organization** → **Invite**
2. Enter team member emails
3. Select their role (Employee, Manager, Admin)
4. Click **Send Invitations**

Invited users will receive an email with a registration link.

## Step 7: Configure Features

1. Go to **Settings** → **Features**
2. Enable/disable features based on your needs:
   - ✅ Time Tracking
   - ✅ Task Management
   - ✅ Invoicing
   - ✅ CRM
   - Toggle others as needed
