# Tutorial: Time Tracking Quickstart

Start tracking time in under 5 minutes.

## Prerequisites

- An active Gauzy account
- At least one project created

## Web App Tracking

### Method 1: Timer Widget

1. Navigate to **Time Tracking** or any page with the timer widget
2. Select a **Project** from the dropdown
3. Optionally select a **Task**
4. Click the **▶ Start** button
5. The timer begins counting
6. Click **⬛ Stop** when done

### Method 2: Manual Time Entry

1. Go to **Time Tracking** → **Add Time**
2. Select:
   - Date and start/end times
   - Project and task
   - Description of work done
3. Click **Save**

### Method 3: Timesheet Entry

1. Go to **Timesheets** → **Weekly**
2. Click on a cell (day × project)
3. Enter hours worked
4. Tab to next cell

## Desktop App Tracking

1. Download the [Desktop Timer](../desktop/desktop-timer)
2. Login with your Gauzy credentials
3. Select a project/task
4. Click Start
5. The app captures:
   - Time duration
   - Screenshots (if enabled)
   - Activity levels (keyboard/mouse)

## Viewing Your Time

| View     | Navigation               |
| -------- | ------------------------ |
| Daily    | Time Tracking → Daily    |
| Weekly   | Timesheets → Weekly      |
| Calendar | Time Tracking → Calendar |
| Reports  | Reports → Time Reports   |

## Tips

- Use keyboard shortcut `Ctrl+Shift+T` to start/stop timer
- Descriptions help with billing and reporting
- Set reminders to start tracking when you begin work
