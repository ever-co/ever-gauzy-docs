# Licensing

Ever Gauzy is available under multiple licenses.

## License Types

| License        | Use Case                 |    Cost    |
| -------------- | ------------------------ | :--------: |
| **AGPL-3.0**   | Open-source, self-hosted |    Free    |
| **Commercial** | SaaS, proprietary use    |    Paid    |
| **Enterprise** | Custom terms, support    | Negotiated |

## AGPL-3.0

The default license for Ever Gauzy is the [GNU Affero General Public License v3.0](https://www.gnu.org/licenses/agpl-3.0.en.html).

### Key Terms

- ✅ Free to use, modify, and distribute
- ✅ Can be used for commercial purposes
- ⚠️ Must disclose source code of modifications
- ⚠️ Must use the same license for derivatives
- ⚠️ Must provide source to users accessing over a network

### AGPL Compliance

If you modify Ever Gauzy and provide it as a network service:

1. You must make your modified source code available
2. Users must be able to access the source
3. The AGPL license must be preserved

## Commercial License

For organizations that cannot comply with the AGPL:

- No source disclosure requirement
- Proprietary modifications allowed
- SaaS usage without AGPL obligations
- Priority support available

Contact [info@ever.co](mailto:info@ever.co) for commercial licensing.
