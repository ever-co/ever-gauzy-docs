# Trademark Policy

Guidelines for using the Ever Gauzy brand and trademarks.

## Trademarks

- **Ever Gauzy** — platform name
- **Ever Co.** — company name
- **Gauzy** — product name
- Platform logos and icons

## Permitted Uses

- ✅ Referring to the software by name in documentation
- ✅ Stating compatibility with Ever Gauzy
- ✅ Using in academic or educational contexts
- ✅ Linking to official Ever Gauzy resources

## Restricted Uses

- ❌ Using "Gauzy" or "Ever" as part of your product name
- ❌ Implying official endorsement
- ❌ Modifying the official logos
- ❌ Using trademarks in domain names for commercial purposes

## Community Forks

When forking Ever Gauzy:

- Must use a distinct name
- Must not use Ever Gauzy branding
- Must credit the original project
- Must comply with AGPL-3.0 license terms

## Contact

For trademark questions, contact [info@ever.co](mailto:info@ever.co).
